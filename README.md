# react-native-esp-wifi-prov

React Native module wrapping esp-wifi-prov

## Installation

```sh
npm install react-native-esp-wifi-prov
```

## Usage

```js
import EspWifiProv from "react-native-esp-wifi-prov";

// ...

const result = await EspWifiProv.multiply(3, 7);
```
